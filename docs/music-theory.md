## 🎹 基础和弦进行

### 1. 卡农进行 (Pachelbel's Canon)
```
I - V - vi - iii - IV - I - IV - V

C 大调:
C - G - Am - Em - F - C - F - G

情感：温暖、怀念、永恒
适用：回忆场景、结局、温馨时刻
```

### 2. 流行进行
```
vi - IV - I - V

C 大调:
Am - F - C - G

情感：希望、向上、励志
适用：标题画面、主角主题、胜利
```

### 3. 爵士 ii-V-I
```
ii - V - I

C 大调:
Dm7 - G7 - Cmaj7

情感：复杂、成熟、都市
适用：咖啡馆场景、NPC 对话、休闲
```

### 4. 史诗进行
```
i - VI - III - VII

Am 小调:
Am - F - C - G

情感：宏大、悲壮、英雄
适用：Boss 战、最终决战、高潮
```

### 5. 悬疑进行
```
i - bII - bVI - bVII

Em 小调:
Em - F - C - D

情感：紧张、不安、神秘
适用：探索未知、恐怖场景、谜题
```

## 🎮 游戏场景音乐指南

### 标题画面
```
目标：吸引玩家，建立基调

建议:
- 中等速度 (90-120 BPM)
-  memorable 旋律
- 完整的主歌 - 副歌结构
- 循环无缝

参考:
- NieR:Automata - Weight of the World
- Persona 5 - Colors Flying High
- Undertale - Megalovania
```

### 探索场景
```
目标：不干扰玩家，营造氛围

建议:
- 慢到中等速度 (70-100 BPM)
- 简单和弦进行
- 重复但有小变化
- 留白空间

参考:
- Zelda:BOTW - 探索音乐
- Skyrim - 环境音乐
- Hollow Knight - 城镇音乐
```

### 战斗场景
```
目标：提升紧张感，配合节奏

建议:
- 快速 (140-180 BPM)
- 强烈节奏
- 不和谐音程增加紧张
- 动态分层 (随战斗强度变化)

参考:
- Devil May Cry - 战斗音乐
- Hades - Boss 战音乐
- Celeste - 追逐场景
```

### 对话场景
```
目标：不抢戏，支持情感

建议:
- 慢速 (60-80 BPM)
- 简单配器
- 低音区为主
- 可静音

参考:
- Persona 系列 - 对话音乐
- Ace Attorney - 询问音乐
```

### 悲伤场景
```
目标：唤起情感共鸣

建议:
- 慢速 (50-70 BPM)
- 小调为主
- 独奏乐器 (钢琴、小提琴)
- 长音符、留白

参考:
- To the Moon - Main Theme
- Final Fantasy 系列 - 哀伤音乐
```

## 🎹 DAW 工作流建议

### 1. 从和弦开始
```
1. 选择和弦进行
2. 用钢琴或 pad 演奏
3. 确定速度和调性
4. 再添加旋律
```

### 2. 从节奏开始
```
1. 设计鼓点节奏
2. 添加贝斯线
3. 再添加和弦
4. 最后旋律
```

### 3. 从旋律开始
```
1. 哼唱或弹奏旋律
2. 配和弦
3. 添加节奏
4. 丰富配器
```
